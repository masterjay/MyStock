📊 數值意義

🔴 散戶極度看多（+6% 以上）
→ 可能接近市場高點
→ 注意風險訊號，考慮減碼
⚪ 散戶中性（-3% 到 +3%）
→ 市場情緒正常
→ 依技術面操作
💡 使用建議
1. 配合技術分析：不單獨作為進出場依據
2. 觀察極端值：特別注意±6%以上的情況
3. 趨勢確認：等待價格突破配合進場
4. 風險控管：設定停損，分批建倉

💰 融資使用率指標
×
📐 計算方式
融資使用率 = 融資餘額 / 整體市場市值 × 100%
反映散戶使用融資買股票的程度
📊 數值意義
• 40%以下：融資水位低，市場偏保守
• 40-50%：融資水位適中
• 50-60%：融資偏高，需注意
• 60%以上：融資過高，風險提升
⚠️ 風險判讀
🟢 融資使用率低（< 45%）
→ 市場籌碼乾淨
→ 上漲空間較大
🔴 融資使用率高（> 60%）
→ 槓桿過高，賣壓沉重
→ 回檔時易引發斷頭賣壓
⚪ 融資使用率適中（45-60%）
→ 市場正常運作
→ 依趨勢操作
💡 使用建議
1. 觀察趨勢：持續上升需警戒
2. 配合指數：高檔+高融資=危險訊號
3. 低檔進場：融資低+指數低=好時機
4. 融資斷頭：大跌時融資戶被迫賣出
📈 期貨多空比指標
×
📐 計算方式
多空比 = 總多單部位 / 總空單部位
包含散戶 + 三大法人的總體部位
📊 數值意義
• > 1.0：市場多單 > 空單，偏多
• = 1.0：多空平衡
• < 1.0：市場空單 > 多單，偏空
• 極端值：> 1.2 或 < 0.8 需注意
🎯 操作策略
🟢 多空比偏低（< 0.9）
→ 空單過多，軋空風險高
→ 若出現上漲，力道可能很強
🔴 多空比偏高（> 1.1）
→ 多單過多，獲利了結壓力
→ 若回檔，賣壓可能較重
⚪ 多空比平衡（0.9-1.1）
→ 市場處於平衡狀態
→ 等待方向明朗
💡 進階觀察
1. 配合外資部位：外資大幅做多/空時特別注意
2. 趨勢轉折：極端值往往是轉折訊號
3. 量價配合：多空比+成交量+價格方向
4. 結算效應：每月第三週三需注意

🎯 MACD 訊號篩選條件

✅ 條件一：多頭股

股價必須在 MA20（20日均線）之上，確認趨勢向上。
不是多頭的股票直接跳過，不搶反彈。

✅ 條件二：日線量縮

近5日均量 / 近20日均量 < 0.85
量縮代表賣壓減緩、籌碼沉澱，是反轉前的訊號。

✅ 條件三：MACD 柱狀體縮小

柱狀體（DIF - MACD）為負值且連續 3 天絕對值在縮小，
或柱狀體剛從負轉正（剛金叉）。代表空方力道減弱中。

✅ 條件四：不離 0 軸太遠

DIF 和 MACD 佔股價的百分比都 < 2.0%
太遠代表還在深度回檔，金叉可能只是反彈而非反轉。

🏷️ 長線 / 短線標記

DIF 和 MACD 都在 0 軸之上 → 長線（多頭格局金叉，可抱久一點）
DIF 或 MACD 在 0 軸之下 → 短線（空方區域金叉，快進快出）

⭐ 訊號強度（1~5 星）

• 量縮比 < 0.6 → +2 星 / < 0.75 → +1 星
• DIF 距 0 軸 < 0.5% → +2 星 / < 1.0% → +1 星
• 柱體縮小速度 > 50% → +1 星
