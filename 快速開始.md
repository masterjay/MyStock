# 台股監控系統 - 快速開始

## 🎯 這個專案是什麼?

一個完整的**台股市場情緒監控系統**，包含:
- 融資使用率監控 (判斷市場過熱/冷卻)
- 台指期多空比 (三大法人部位分析)
- 外資淨部位追蹤
- 30-60天歷史趨勢圖表
- 專業的金融終端機風格介面

## ⚡ 10秒快速開始

### 方法1: 只看介面 (最快)
```bash
# 直接用瀏覽器開啟
open dashboard.html
```
會顯示模擬數據的專業監控面板

### 方法2: 抓真實數據 (完整版)

**Step 1: 安裝 Python 套件**
```bash
cd backend
pip install -r requirements.txt
```

**Step 2: 執行數據收集**
```bash
python data_collector.py
```

會出現選單:
```
選擇執行模式:
1. 收集最近交易日的數據 (預設)
2. 收集過去 30 天的歷史數據  ← 推薦!
3. 收集過去 60 天的歷史數據
4. 啟動定時任務 (每天 15:30 自動執行)
```

**建議選擇 2 或 3** - 一次抓取完整歷史數據!

**Step 3: 開啟網站**
```bash
# 用瀏覽器開啟
open dashboard.html
```

## 🔥 新功能亮點

### 1. 歷史數據回補
即使今天是假日,也能抓取過去交易日的數據:
- 自動跳過週末
- 自動找最近的交易日
- 一次抓取 30-60 天完整資料

### 2. 智能避免重複
- 已有的資料不會重複抓取
- 節省時間和網路請求

### 3. 互動式選單
執行時可以選擇:
- 只抓最新的
- 補齊 30 天
- 補齊 60 天
- 啟動定時任務

## 💡 使用建議

### 第一次使用
```bash
cd backend
pip install -r requirements.txt
python data_collector.py
# 選擇 2 或 3，抓取完整歷史數據
```

### 之後每日更新
```bash
python data_collector.py
# 選擇 1 或直接 Enter，只抓最新的
```

### 自動化 (設定後不用管)
```bash
python data_collector.py
# 選擇 4，啟動定時任務
# 每天下午 3:30 自動執行
```

## 📱 網站功能

### 主面板
- 即時融資使用率 (自動變色警示)
- 台指期多空比
- 外資淨部位
- 30天趨勢圖表

### 數據解讀面板
點擊右上角「📖 數據解讀」按鈕，會彈出完整的:
- 融資使用率解讀 (高/中/低水位)
- 多空比解讀 (偏多/平衡/偏空)
- 外資部位解讀
- 綜合判斷要點

## ⚠️ 注意事項

1. **交易日限制**: 只有交易日才有數據
2. **今天是聖誕節**: 建議抓過去 30 天的歷史數據
3. **請求頻率**: 程式有設定間隔，避免被擋
4. **僅供參考**: 這是輔助指標，不是進出場訊號

## 🆘 常見問題

**Q: 執行後顯示「無法取得數據」?**
- 今天可能是假日，選擇選項 2 或 3 抓歷史數據

**Q: 網站還是顯示模擬數據?**
- 確認 `data/market_data.json` 是否存在
- 重新整理瀏覽器 (Ctrl+F5 或 Cmd+Shift+R)

**Q: 想要更多天的數據?**
- 修改 data_collector.py，把 30 或 60 改成你想要的天數

**Q: 可以在手機上看嗎?**
- 可以! 網站是響應式設計，手機也能正常顯示
